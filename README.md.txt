# Temperature Converter in C

## Description

This is a simple C program that converts temperatures between different units using a menu-driven approach.

## Features

* Celsius to Fahrenheit
* Fahrenheit to Celsius
* Celsius to Kelvin
* Kelvin to Celsius
* Simple and easy-to-understand code
* User-friendly menu

## Technologies Used

* C Programming Language
* GCC Compiler (or any standard C compiler)

## How to Run

1. Save the source code as `main.c`.
2. Compile the program:

   ```
   gcc main.c -o temp_converter
   ```
3. Run the program:

   ```
   ./temp_converter
   ```

   On Windows:

   ```
   temp_converter.exe
   ```

## Sample Output

```

2. Fahrenheit to Celsius
3. Celsius to Kelvin
```
